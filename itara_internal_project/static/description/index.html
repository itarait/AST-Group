Project Management App

Features:

  • In Sale Order Line new field is added - Due Date
  • The due date which is given in sale order line item will be the deadline date of the particular project task
  • In Sales Analysis report, we can filter the data's by due date and also Project Description
  • Stages of the project is added for project - Inprogress - Hold - Closed
  • In project report we can see stage of the project
